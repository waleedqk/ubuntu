# vim Navigate About a File

**Move about a file**

    h, j, k, l

**Move to top of file**

    H

**Move to bottom of screen**

    L

**Jump to the start of the line**

    0

**Jump to the end of the line**

    $

**Go to the first line of the document**

    gg

**Go to the last line of the document**

    G

**Go to line 5**

    5G
